第一步

先用 Add Folder 開啟備課工作目錄

Antigravity 的能力從資料夾開始。你開哪個資料夾，它就在哪個任務現場工作。

Add Folder / Open Folder
選取研習資料夾
左側檔案樹出現教材與範例檔

指令技巧一

用 @ 精準鎖定檔案

輸入 @ 叫出檔案清單，直接指定要分析或修改的本機檔案。

@
請分析 @學生成績狀況表.xlsx

好處：避免 AI 找錯檔案，任務目標更清楚。

指令技巧二

用 / 召喚快捷指令與工作流

斜線像快捷菜單，可以叫出工作流、瀏覽器、目標模式與對齊訪談。

/
/goal
/browser
/grill-me

安全把控

二廚放鹽，主廚嘗味

當 AI 準備執行程式或修改檔案時，Antigravity 會停下來請你核准。

準備執行：python generate_report.py
Approve 同意
Deny 拒絕

這就是 Human-in-the-Loop 人機協同。

Security Preset

研習現場保持 Default 最穩

安全預設控制 Agent 能在哪裡活動，以及什麼時候需要先問你。

Default
工作目錄防護模式，研習推薦值。
Full Machine
可讀取更大範圍，但執行指令仍需核准。
Turbo Mode
速度快但風險高，研習請勿開啟。

Artifact Review Policy

文件出菜前，要不要每盤必審？

Always Ask 適合研習與初學者；Always Proceed 適合熟悉流程後追求效率。

Always Ask
每盤必審
Always Proceed
信任直出
品質把關 vs 速度效率
初學建議：先 Always Ask

避坑指南

檔案要關好，資料夾要乾淨

正在開啟的 Word / Excel 可能鎖檔；過多無關素材會拖慢 Agent 判斷。

  • 修改前先關閉 Office 檔案視窗。
  • 工作目錄只放當前任務需要的檔案。
  • 巨大影片、照片、舊 PDF 先移出去。

第二章總結

會開資料夾、會指定檔案、會看懂核准，就能開始實作

  • 先用 Add Folder 載入今天的備課資料夾。
  • 用 @ 鎖定檔案，用 / 召喚工作流。
  • 維持 Default，避免 Turbo Mode，重要動作先 Approve。
