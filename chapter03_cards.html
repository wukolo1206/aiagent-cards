學習目標

第三單元核心練習與體驗目標

老師將在此實戰中體驗 AI Agent 的四大核心能力，實現全自動辦公減量：

1. ReAct 自主規劃
老師給予最終目標，AI 會自動規劃與拆解工作步驟。
2. 本機工具呼叫
主動呼叫工具讀取 Excel 檔案，並背景執行 Python 產出排版美觀的 Word 檔案。
3. 人機協同防線
敏感或寫檔操作會自動暫停，由人類點擊 Approve 始執行。
4. 自動自我糾錯
程式報錯或缺少套件時，AI 能自主發現問題並自動修正重跑。
